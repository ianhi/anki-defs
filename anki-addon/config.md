## anki-defs Settings

- **aiProvider**: AI provider to use (`claude`, `gemini`, or `openrouter`)
- **claudeApiKey**: Your Anthropic API key
- **geminiApiKey**: Your Google Gemini API key
- **geminiModel**: Gemini model to use (default: `gemini-2.5-flash-lite`)
- **openRouterApiKey**: Your OpenRouter API key
- **openRouterModel**: OpenRouter model to use (default: `google/gemini-2.5-flash`)
- **showTransliteration**: Include romanized transliteration in definitions
- **defaultDeck**: Default Anki deck for new cards
- **defaultModel**: Default note type for new cards
- **fieldMapping**: Maps card data fields (Word, Definition, Example, Translation) to your note type's field names
- **port**: HTTP server port (default: 28735)
